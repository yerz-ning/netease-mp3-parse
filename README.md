# 网易云音乐 MP3 解析器

一个网页的网易云音乐MP3解析工具，输入网易云音乐歌曲ID或完整分享链接，即可播放/下载MP3。

🌐 **全平台支持**：安卓 / iOS / Windows / macOS / Linux —— 只要可以使用浏览器就能用（下载需浏览器支持下载功能）

## ✨ 功能特点

- 🎯 输入歌曲ID或完整分享链接，一键解析
- ⚡ 可直接播放MP3或下载
- 📱 纯HTML

## 🎵 在线使用（无需手动部署）
🔗 [点此打开已完成部署的工具](https://netease-mp3-parse.yerz.qzz.io/)

## 📌 使用方法

1. 打开网易云音乐APP/网页，找到你想下载的歌曲
2. 复制歌曲分享链接
3. 粘贴到输入框，点击解析即可使用
